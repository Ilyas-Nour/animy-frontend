## 🚀 Overview

**Animy** is a high-performance, modern web application designed for anime enthusiasts. It provides a unified interface to discover, stream, and interact with a vast library of anime and manga content. Built with a focus on user experience, it features smooth animations, real-time interactions, and a premium aesthetic.

## ✨ Key Features

- 🎭 **Stunning UI/UX**: Powered by Framer Motion and GSAP for fluid, cinema-like transitions and micro-animations.
- 📺 **Advanced Video Player**: Custom integration with `Vidstack` and `HLS.js` supporting multi-resolution streaming and subtitles.
- 💬 **Real-time Community**: Live chat and notification system powered by `Socket.io`.
- 📖 **Manga Reader**: A clean, distraction-free reading experience for manga lovers.
- 🔍 **Smart Search**: Fast and intuitive search functionality to find any series instantly.
- 👤 **Personalized Profiles**: Track your progress, maintain watchlists, and connect with friends.
- 📊 **Anime Analytics**: Detailed stats and charts using `Recharts`.
- 🌐 **Responsive Design**: Optimized for everything from mobile phones to ultra-wide monitors.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/), [Shadcn UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/), [GSAP](https://greensock.com/gsap/)
- **State Management**: [React Context API](https://react.dev/learn/passing-data-deeply-with-context)
- **Networking**: [Axios](https://axios-http.com/)
- **Form Handling**: [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Smooth Scrolling**: [Lenis](https://lenis.darkroom.engineering/)

## ⚙️ Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ilyas-Nour/animy-frontend.git
   cd animy-frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env.local` file in the root directory and add:
   ```env
   NEXT_PUBLIC_API_URL=http://localhost:3001/api/v1
   NEXT_PUBLIC_SOCKET_URL=http://localhost:3001
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open the application**
   Navigate to [http://localhost:3000](http://localhost:3000)
